## Sobre o projeto

Site estático desenvolvido com **HTML, CSS e JavaScript puro**, sem frameworks ou build tools. A interface segue um tema escuro com efeitos de glassmorphism, animações de scroll e layout responsivo para desktop e mobile.

Atualmente estou em transição para a área de **Análise de Dados**, construindo projetos práticos com Python, SQL e visualização de dados. O portfólio reflete esse foco de forma transparente — incluindo o que já domino e o que estou aprendendo ativamente.

### Seções do site

| Seção | Conteúdo |
|-------|----------|
| **Inicio** | Apresentação, tags de tecnologia e links para GitHub, LinkedIn e contato |
| **Sobre** | Perfil, paixão por tecnologia e foco em dados |
| **Habilidades** | Stack em programação, dados e ferramentas |
| **Jornada** | Linha do tempo da trilha de aprendizado (ADS → Python → SQL → ETL) |
| **Projetos** | Estudos de caso em desenvolvimento, com link para o GitHub |
| **Aprendizado** | Tópicos em progresso (Python, SQL, Power BI, Estatística, ETL…) |
| **Contato** | E-mail, GitHub, LinkedIn e disponibilidade remota |

## Funcionalidades

- Navegação fixa com destaque automático da seção ativa ao rolar a página
- Menu mobile com animação hamburger
- Animações de entrada com `IntersectionObserver`
- Cursor customizado em dispositivos com mouse (desativado em touch/mobile)
- Botão "voltar ao topo" após scroll
- Scroll suave entre âncoras
- Layout totalmente responsivo
- Tema escuro com gradientes e grid de fundo
